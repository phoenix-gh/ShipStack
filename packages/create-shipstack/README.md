# create-shipstack-app

Create a new ShipStack TanStack Start app.

## Usage

```bash
pnpm create shipstack-app my-app
cd my-app
pnpm install
pnpm dev
```

The generated app includes Cloudflare Workers configuration, health routes,
API route foundations, CI and deploy workflows, and a generated `AGENTS.md`.

## Status

`0.1.0-alpha.1` is a release candidate package for the first ShipStack MVP.
